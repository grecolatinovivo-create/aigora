# Regola d'oro Git

## USA SOLO GITHUB DESKTOP. MAI IL TERMINALE.

Punto. Fine.

---

## Perché

Terminal e GitHub Desktop usano lo stesso repo. Se girano in parallelo si bloccano a vicenda con lock file e il push si inceppa per ore.

## Flusso corretto

1. Fai le modifiche al codice
2. Apri GitHub Desktop
3. Scrivi il messaggio di commit
4. Clicca **Commit to main**
5. Clicca **Push origin**

Basta.

## Se GitHub Desktop sembra bloccato

Aspetta. Non aprire il Terminal. Non cliccare niente. Aspetta.

## Se Claude ti dà un comando git da Terminal

Ignoralo. Digli: *"fallo tu dal sandbox o aspetta GitHub Desktop"*.
